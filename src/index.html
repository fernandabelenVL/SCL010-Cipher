Bienvenido a Little Secrets

Bienvenidos a Little Secrets, una aplicación que te permitirá crear mensajes secretos para enviar a tus amigas.
En ella podrás escribir un mensaje en clave y también decodificar los mensajes que te envíen, así todo quedará entre las personas que quieras.
Para comenzar necesitas un “número clave” el que servirá tanto para enviar como para recibir tu mensaje secreto, este número clave creará un mensaje en código y para que tu amiga lo pueda leer necesita el mismo número para descodificarlo, así que es importante que se pongan de acuerdo cuál es el número que usarán para contar sus secretos.

Instrucciones

  • 1. Ingresa un número que funcionará de "clave" para codificar o decodificar tu mensaje.
  • 2. Escribe el mensaje que quieres que sea secreto, o ingresa el mensaje que quieres descodificar.
  • 3. Haz click en los botones "Codificar" o "Decodificar" para mostrar el resultado de tu mensaje.
  • ¡Y ya está! ya tienes el resultado de tu mensaje secreto, ahora solo queda copiarlo para enviárselo a tus amigas.
